## 2. Описание тестируемого продукта

Библиотека `geometric_lib` представляет собой набор модулей для вычисления площади и периметра различных геометрических фигур:

- **circle.py** - вычисление площади и периметра круга
  - Функция `area(r)`: вычисляет площадь круга по формуле S = πR²
  - Функция `perimeter(r)`: вычисляет периметр круга по формуле P = 2πR

- **rectangle.py** - вычисление площади и периметра прямоугольника
  - Функция `area(a, b)`: вычисляет площадь прямоугольника по формуле S = ab
  - Функция `perimeter(a, b)`: вычисляет периметр прямоугольника по формуле P = 2(a + b)

- **square.py** - вычисление площади и периметра квадрата
  - Функция `area(a)`: вычисляет площадь квадрата по формуле S = a²
  - Функция `perimeter(a)`: вычисляет периметр квадрата по формуле P = 4a

- **triangle.py** - вычисление площади и периметра треугольника
  - Функция `area(a, h)`: вычисляет площадь треугольника по формуле S = ah/2
  - Функция `perimeter(a, b, c)`: вычисляет периметр треугольника по формуле P = a + b + c

## 4. Стратегия тестирования

### Типы тестирования:
- **Функциональное тестирование**: проверка соответствия функций их математическим формулам
- **Тестирование граничных значений**: проверка работы с нулевыми значениями и единичными значениями
- **Тестирование типов данных**: проверка работы с целыми и дробными числами

### Методы тестирования:
- **Unit-тестирование**: изолированное тестирование отдельных функций
- **Автоматизированное тестирование**: использование фреймворка unittest для автоматического запуска тестов

### Техники тестирования:
- **Эквивалентное разбиение**: тестирование нормальных значений, граничных значений и особых случаев
- **Анализ граничных значений**: проверка работы с нулевыми и минимальными значениями
- **Таблицы решений**: проверка различных комбинаций входных параметров

## 5. Критерии приемки

Тестирование считается успешным, если:
