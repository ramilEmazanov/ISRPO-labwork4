# Geometric Library

Простая и удобная Python-библиотека для вычисления площади и периметра основных геометрических фигур.

## Описание

Библиотека `geometric_lib` представляет собой набор модулей для вычисления площади и периметра различных геометрических фигур:

- **circle.py** - вычисление площади и периметра круга
  - Функция `area(r)`: вычисляет площадь круга по формуле S = πR²
    - Валидация: проверяет, что радиус r ≥ 0, иначе выбрасывает ValueError
  - Функция `perimeter(r)`: вычисляет периметр круга по формуле P = 2πR
    - Валидация: проверяет, что радиус r ≥ 0, иначе выбрасывает ValueError

- **rectangle.py** - вычисление площади и периметра прямоугольника
  - Функция `area(a, b)`: вычисляет площадь прямоугольника по формуле S = ab
    - Валидация: проверяет, что длина a ≥ 0 и ширина b ≥ 0, иначе выбрасывает ValueError
  - Функция `perimeter(a, b)`: вычисляет периметр прямоугольника по формуле P = 2(a + b)
    - Валидация: проверяет, что длина a ≥ 0 и ширина b ≥ 0, иначе выбрасывает ValueError

- **square.py** - вычисление площади и периметра квадрата
  - Функция `area(a)`: вычисляет площадь квадрата по формуле S = a²
    - Валидация: проверяет, что сторона a ≥ 0, иначе выбрасывает ValueError
  - Функция `perimeter(a)`: вычисляет периметр квадрата по формуле P = 4a
    - Валидация: проверяет, что сторона a ≥ 0, иначе выбрасывает ValueError

- **triangle.py** - вычисление площади и периметра треугольника
  - Функция `area(a, h)`: вычисляет площадь треугольника по формуле S = ah/2
    - Валидация: проверяет, что основание a ≥ 0 и высота h ≥ 0, иначе выбрасывает ValueError
  - Функция `perimeter(a, b, c)`: вычисляет периметр треугольника по формуле P = a + b + c
    - Валидация: проверяет, что все стороны a, b, c ≥ 0, иначе выбрасывает ValueError
Библиотека включает полный набор unit-тестов, обеспечивающих корректность работы всех функций. Все функции содержат валидацию входных данных для защиты от некорректных значений.

Просто скопируйте файлы в ваш проект или клонируйте репозиторий:


## Математические формулы

### Площадь
- **Круг:** S = πR²
- **Прямоугольник:** S = ab
- **Квадрат:** S = a²
- **Треугольник:** S = ah/2

### Периметр
- **Круг:** P = 2πR
- **Прямоугольник:** P = 2a + 2b
- **Квадрат:** P = 4a
- **Треугольник:** P = a + b + c
